GIT Flow

GIT CLONE-для того, щоб "склонувати" репоиторій
GIT PULL-для того,щоб отримати зміни з репозиторію
GIT CHECKOUT -B BRANCHNAME-створити нову гілку та перейти на неї
GIT ADD .-додати зміни, які зроблено локально
GIT COMMT -M "COMMIT MESSEGE"-підписати зміни
GIT PUSH ORIGIN BRANCHNAME-завантажити на git server
